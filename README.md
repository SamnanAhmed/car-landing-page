Aventador Landing Page 🏎️

A fully responsive Lamborghini Aventador-themed landing page built with HTML, CSS, and vanilla JavaScript.

🔗 Live Demo: samnan-car-website.netlify.app

About

This project was built by following along with tutorials and then extending the design with custom fixes and improvements. The goal was to practice real-world frontend skills — responsive layouts, CSS animations, and DOM manipulation — without relying on any frameworks or libraries.

Features


Fully responsive design — works smoothly across mobile, tablet, and desktop
Mobile-first navbar with a hamburger menu, overlay, and scroll lock
CSS Grid & Flexbox for layout structure
Custom CSS transforms for the angled, skewed video and image cards
Scroll-spy navigation — the active nav link updates automatically as you scroll through sections
Hover-reveal car model cards with smooth CSS transitions
Scroll-triggered animations for section reveals
CSS custom properties (variables) for consistent theming and easy color/style updates


Sections


Home — Hero section with title and call-to-action button
About — Brand description with a skewed video showcase
Models — Car model cards (Silver, Yellow, Blue Aventador) with hover effects
Information — Technical specs (power, top speed, 0–100 km/h)
Contact — Newsletter subscription form
Footer — Links and social media


Tech Stack


HTML5
CSS3 (Grid, Flexbox, Custom Properties, Transforms, Transitions)
Vanilla JavaScript (DOM manipulation, event listeners, scroll-based logic)


What I Learned

Building this project helped me go beyond just following a tutorial — I spent time debugging real issues like CSS specificity conflicts, transform stacking order, responsive breakpoints, and writing JavaScript for scroll-based active link highlighting. It pushed me to understand why the code works, not just copy it.
